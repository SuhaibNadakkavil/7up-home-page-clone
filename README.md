## Overview
This project is a front-end UI clone of the official 7UP website. ## Features
- Fully structured website layout inspired by 7UP's official design  
- Fixed header with navigation links and social media icons  
- Hero banner with bold promotional text  
- Recipes section styled with centered overlay text  
- Store locator promotional section  
- Footer with complete policy and navigation links  
- Clean and organized CSS  
- Beginner-friendly and front-end-focused (no frameworks)
